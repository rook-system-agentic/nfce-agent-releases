# NFC-e Capture Agent

Agente de captura automática de NFC-e do PDV para o Rook System.

## Download

Acesse a seção [Releases](https://github.com/rook-system-agentic/nfce-agent-releases/releases) para baixar a versão mais recente.

## Instalação

1. Baixe o executável para seu sistema operacional (Windows ou Mac)
2. Execute o instalador
3. Acesse app.rooksystem.com.br > Integrações > NFC-e Capture Agent
4. Gere seu token e cole no agente
5. Indique a pasta de XMLs do seu PDV
6. Pronto — o Rook receberá suas vendas automaticamente
